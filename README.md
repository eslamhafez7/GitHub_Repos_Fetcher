# GitHub Repositories Fetcher

A simple web application that allows users to input a GitHub username and retrieve a list of that user's repositories. Repositories are displayed in descending order based on their last update time. Users can trigger the retrieval by clicking a button or pressing the Enter key after entering the username.

## Features

- Enter a GitHub username to fetch repositories.
- Click a button or press Enter to trigger fetching.
- Display repositories with their names, URLs, and star counts.

## Technologies Used

- HTML
- CSS
- JavaScript
- Fetch API
- SweetAlert library (https://sweetalert2.github.io/)
